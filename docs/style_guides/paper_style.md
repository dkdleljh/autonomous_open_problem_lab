# 논문 스타일 가이드

## 1. 필수 섹션

1. 제목
2. 초록
3. 배경
4. 관련 연구
5. 정의와 표기
6. 핵심 아이디어
7. 보조정리
8. 주정리
9. 증명
10. 계산 검증
11. 한계
12. 재현성
13. 참고문헌
14. 부록

## 2. 번호 규칙

- 한국어, 영어 문서의 정리 번호 동기화 필수
- 수식 번호 동기화 필수
- 참고문헌 번호 동기화 필수

## 3. 논리 공백 방지 규칙

1. 각 정리마다 가정과 결론을 분리해 적는다.
2. proof DAG 노드와 본문 문장을 연결한다.
3. 계산 검증의 범위, seed, 시간, 제한을 명시한다.
4. 형식화 실패 구간을 부록에 기록한다.

## 4. 표현 규칙

- 해결 주장 문장은 검증 통과 상태에서만 사용한다.
- 계산 실험은 실험적 지지로 표현한다.
- 과장 표현을 금지한다.
