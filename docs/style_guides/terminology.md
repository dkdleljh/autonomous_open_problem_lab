# 용어 가이드

## 1. 한영 용어집

- 난제: open problem
- 보조정리: lemma
- 주정리: main theorem
- 증명 그래프: proof DAG
- 형식화: formalization
- 반례: counterexample
- 재현성: reproducibility
- 참고문헌: references
- 의무 항목: obligation

## 2. 금지 표현

- 자명하다
- 생략한다
- 유사하게
- 당연하다

위 표현은 논리 공백을 유발하므로 문서와 논문에서 사용하지 않는다.

## 3. 문체 규칙

1. 근거 중심 서술
2. 조건, 범위, 한계를 명시
3. 주장과 증거를 분리
4. 계산 결과는 실험적 지지로만 표기

## 4. 한자 금지 규칙

- 문서, 주석, 로그 메시지에서 한자 사용 금지
- 한자가 포함된 외부 문자열은 가능한 범위에서 한글 또는 영어로 치환
