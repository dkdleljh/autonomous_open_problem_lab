# 실험 정책

## 1. 목적

계산 실험은 증명 보조 자료로만 사용하며 일반 명제 해결의 직접 근거로 사용하지 않는다.

## 2. 필수 기록 항목

- 문제 식별자
- 실행 시각
- 탐색 범위
- seed
- 실행 시간
- 사용 엔진 버전
- 결과 분류

## 3. 저장 위치

- `data/experiments/<problem_id>_counterexample.json`
- `data/audit_logs/pipeline_audit.jsonl`

## 4. 금지 사항

- 범위 미기록 실험
- seed 미기록 실험
- 결과만 기록하고 조건을 생략하는 실험
- 실험 결과를 해결 주장으로 승격하는 행위

## 5. 승인 기준

실험 결과는 verifier를 통과하고 재현성 부록에 반영될 때만 논문 본문에서 인용 가능하다.
