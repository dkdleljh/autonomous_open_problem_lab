# 인용 정책

## 1. 기본 원칙

모든 주장에는 출처를 연결한다. 출처 없는 서술은 검증 단계에서 경고 또는 실패 대상으로 처리한다.

## 2. 필수 규칙

1. 참고문헌은 BibTeX로 생성한다.
2. 한국어와 영어 논문의 참고문헌 번호는 동일해야 한다.
3. 중복 URL 또는 충돌 항목은 verifier 경고에 기록한다.

## 3. 최소 메타데이터

- 제목
- URL 또는 DOI
- 연도
- 출처 유형

## 4. 금지 사항

- 근거 없는 일반화
- 출처 누락
- 인용 번호 불일치

## 5. 자동 검사

Paper QA는 참고문헌 키의 존재 여부를 검사하며 누락 시 차단한다.
