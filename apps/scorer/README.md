# Scorer 앱

`aopl/apps/scorer.py`는 정규화된 문제를 우선순위 관점에서 평가하는 단계다.

## 역할

- 형식화 가능성, 분해 가능성, 실험 가능성, 논문화 용이성 같은 지표를 계산한다.
- 선택 여부와 근거를 score card로 남긴다.
- 이후 counterexample과 proof 단계의 대상을 좁힌다.

## 주요 출력

- `data/normalized/*_score.json`
- 선택 여부와 세부 항목별 점수

## 운영 팁

- 점수는 절대적 진실이 아니라 선별 기준이다.
- 특정 문제군이 계속 과대평가되면 가중치와 입력 정규화 상태를 함께 점검해야 한다.
